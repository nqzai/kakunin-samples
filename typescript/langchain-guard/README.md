# Kakunin TypeScript LangChain Sample

This sample demonstrates how to enforce pre-flight scope validation and machine certificate checking on LangChain JS/TS tools using `@kakunin/langchain`.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the sample:
   ```bash
   KAKUNIN_API_KEY="your_api_key" OPENAI_API_KEY="your_key" npm start
   ```
