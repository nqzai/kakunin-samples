# Kakunin + Mastra Compliance Tools

This directory contains a TypeScript sample demonstrating how to integrate Kakunin compliance, X.509 certificate checks, and immutable behavioral event logs with Mastra agents.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set your Kakunin API Key:
   ```bash
   export KAKUNIN_API_KEY="your-kakunin-api-key"
   ```

3. Run the script:
   ```bash
   npm start
   ```

## Integration Code Example

Pass the tools generated by `KakuninIntegration` directly into your Mastra agent configuration:

```typescript
import { KakuninIntegration } from '@kakunin/mastra';
import { Agent } from 'mastra';
import { openai } from '@ai-sdk/openai';

const kakunin = new KakuninIntegration({
  apiKey: process.env.KAKUNIN_API_KEY,
});

const agent = new Agent({
  name: 'ComplianceAgent',
  model: openai('gpt-4o'),
  instructions: 'You are a compliance-governed trading agent.',
  tools: kakunin.getTools(), // verifyAgentCertificate, checkAgentScope, etc.
});
```
